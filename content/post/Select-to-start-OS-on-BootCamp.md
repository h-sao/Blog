macOS に BootCamp を入れると、デフォルト起動OSが Windows になります  
（わたしの環境ではそうなりました）

これ、デフォルトを macOS に戻すには、  

> システム環境設定  
　＞ 起動ディスク  

ここを macOS にすればOK

![](/pic/Select-to-start-OS-on-BootCamp_00.png)

Windows を立ち上げたい時には、電源を入れた直後に `option` を長押しすれば、起動OS選択画面になります
